# Deploy to Firebase

## On your PC

### 1. Unzip and enter folder
```
unzip project-reality-firebase.zip
cd project-reality-release
```

### 2. Install project + Firebase CLI
```
npm install
npm install -g firebase-tools
```

### 3. Login to Google
```
firebase login
```

### 4. Deploy
```
firebase deploy
```

### 5. View your app
https://project-reality-8966a.web.app
